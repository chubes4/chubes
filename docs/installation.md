# Installation

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher

## Installation Steps

1. Download the theme ZIP file from the releases page
2. In WordPress admin, go to Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Upload the `chubes.zip` file
5. Activate the theme

## Recommended Plugins

For full functionality, install and activate these plugins:

- **chubes-contact**: Provides contact form functionality
- **chubes-docs**: Adds documentation and project tracking features
- **chubes-games**: Enables game-related post types and blocks

## Theme Setup

After activation:

1. Go to Appearance > Customize to configure homepage content
2. Set up your homepage by creating a page and assigning it as the front page
3. Configure navigation menus under Appearance > Menus
4. Add content to your journal posts and pages

## File Structure

- `/inc/core/templates/` - Theme templates
- `/inc/core/assets.php` - Asset enqueuing
- `/inc/journal/` - Journal post type registration
- `/assets/css/` - Stylesheets
- `/assets/js/` - JavaScript files
